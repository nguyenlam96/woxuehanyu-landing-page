Foreground Service Media Playback

This page is provided for Google Play policy review of the FOREGROUND_SERVICE_MEDIA_PLAYBACK permission used by Wo xue Hanyu.

Declared task: Media playback

Use case: The app plays user-requested learning audio and other lesson media, and playback may continue while the user is no longer directly interacting with the app.

Review material: The video below demonstrates the in-app media playback flow for this permission.

Policy Description

  • The permission is used only for foreground media playback started by the user.
  • The playback task is noticeable to the user because audio continues while the app is backgrounded or the user switches away from the current screen.
  • This page hosts the demonstration video requested for Google Play review.

If playback does not start in your browser, use the direct video file link below.
